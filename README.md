# Senior Capstone Attendance App

The Senior Capstone Attendance App is a web and mobile application designed to make student attendance tracking easier for scheduled events.

The purpose of this project is to replace manual attendance tracking with a faster QR code check-in process. Staff members can create events and generate QR codes, while students can scan those QR codes from the mobile app to record their attendance.

## Project Purpose

This project helps schools or organizations track student attendance at events in a simple and organized way.

Instead of using paper sign-in sheets or manually entering attendance, this system allows students to check in by scanning a QR code connected to a specific event.

## Web App

The web app is used by staff, teachers, and administrators.

It allows them to:

- Log in securely
- Create new events
- Edit existing events
- Add and manage students
- Generate QR codes for events
- Print QR codes for students to scan
- View attendance records for events

## Mobile App

The mobile app is used by students.

It allows students to:

- Log in securely
- Scan an event QR code
- Check in to an event
- Add the event to their attendance record
- Confirm that their attendance was recorded

## How It Works

1. A staff member creates an event in the web app.
2. The web app generates a QR code for that event.
3. The QR code is printed or displayed at the event.
4. A student logs into the mobile app.
5. The student scans the QR code.
6. The student is checked in to the event.
7. The attendance record is saved and can be viewed by staff.

## Summary

This project provides a simple way to manage event attendance using QR codes. The web app gives staff control over events, students, and attendance records, while the mobile app gives students an easy way to check in to events.
